# 遥远，记忆

这是一个保存着珍贵聊天记录和回忆的项目，记录了那些遥远而美好的记忆片段。

## 项目简介

本项目包含了：
- 聊天记录的HTML格式保存
- 搜索功能，支持快速查找历史消息
- 头像显示和用户信息展示
- 按日期组织的消息记录

## 功能特性

### 🔍 智能搜索
- 支持关键词搜索历史聊天记录
- 智能过滤发送者名称，避免无效结果
- 高亮显示搜索匹配内容

### 👤 用户头像
- 自动显示聊天参与者的头像
- 优雅的头像布局和样式
- 支持头像加载失败时的降级处理

### 📅 时间线浏览
- 按日期组织的聊天记录
- 清晰的时间显示格式
- 便捷的页面导航

## 技术实现

- **前端**: HTML5, CSS3, JavaScript
- **搜索**: 原生JavaScript实现
- **样式**: 响应式设计，现代化UI
- **数据结构**: HTML格式的聊天记录存储

## 使用说明

1. 打开 `Pages/search.html` 进行搜索
2. 在搜索框中输入关键词
3. 查看搜索结果，包含头像、发送者、时间和内容
4. 点击结果可以跳转到对应的聊天记录页面
